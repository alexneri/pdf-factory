# Chapter 1: Introduction

This is the first chapter of our multi-chapter example.

## What is This?

This example demonstrates how pdf-factory can combine multiple markdown files into a single, cohesive PDF document.

## Key Concepts

- Multiple files can be organized in a directory
- Files are processed in alphabetical order
- All headings are included in a unified table of contents
