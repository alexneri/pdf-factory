# Example Document

This is an example markdown document to demonstrate the pdf-factory CLI.

## Introduction

Welcome to **pdf-factory**! This tool allows you to create professional PDFs from Markdown files.

### What is Markdown?

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents.

## Features

The pdf-factory provides several powerful features:

- **Single file publishing**: Convert individual markdown files to PDF
- **Directory publishing**: Combine multiple markdown files into a single PDF
- **Custom templates**: Use HTML templates for full design control
- **Table of contents**: Automatically generated from your headings
